# 反拖堂程序 2.0
班级老登每次在吃饭前的最后一节课总是拖堂，导致抢不到饭，所以一怒之下开发了这个程序

这是一个基于 Windows + Tkinter 的课堂管理工具，用于在下课时间提醒、干预拖堂行为，并提供倒计时与强制反馈机制。
根据实测，老师对全屏窗口绘制的接受度普遍不高，所以最终还是采取了窗口化的形式

-附加了一些新的代码来防止老师关闭窗口（考虑到老师操作电脑的水平没有做进程保护）

---
## 使用说明
建议使用Windows任务计划器来定时启动
大概在下课时间点前的2分15秒启动即可，因为启动有延迟


## ✨ 功能特性

- ⏰ **多阶段下课提醒机制**
  - 提前 2 分钟提醒
  - 二次确认（下课 / 拖堂）
  - 最终密码验证（默认密码为LZ,修改方法见配置说明）

- 🧠 **行为干预设计**
  - 窗口强制置顶
  - 失焦自动拉回 + 抖动提示
  - 系统提示音反馈

- 🪟 **现代 UI 动效**
  - Win11 风格玻璃倒计时胶囊
  - 平滑缩放 / 展开动画
  - 半透明 + 鼠标穿透

- 🔒 **强制结束机制**
  - 密码解锁关闭程序
  - 一键调用 Windows 滑动关机界面

## 🚀 使用方法

### 方式一：直接运行源码

    python anti-class.py

### 方式二：打包为 EXE（推荐）

使用 PyInstaller：

    pyinstaller -w -F anti-class.py

参数说明：
- `-w`：无控制台窗口  
- `-F`：单文件打包

## ⚙️ 配置说明

在源码中可修改：

PASSWORD = "LZ"  # 关闭程序密码

SLIDE_EXE = r"C:\Windows\System32\SlideToShutDown.exe"
